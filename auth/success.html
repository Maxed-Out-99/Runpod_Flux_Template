Need help choosing a download option?

Download All Models (FP8):
Best for general use. UNETs are lower precision (FP8), which reduces file size and memory use.

Download All Models (FP16):
Includes full precision UNETs (FP16). Only recommended if your pod has more than 24 GB VRAM (eg. 5090) and you specifically want max quality.

Download Only Small Models:
Fastest. This skips the large UNETs (Canny, Depth, Fill). You can download models as needed in ComfyUI using the "(Down)Load UNET MXD" node.
